SketchGPT: Animate static 2D sketch illustrations using text-to-video prior

Anonymous Authors
Paper Code

Abstract. Animation of the sketches using traditional tools is challenging and complex. The sketches provide a visual representation for an explanation, and the animation of these sketches provides an experience of real-time scenarios. In this paper, we propose SketchGPT, a method to animate a given input sketch using the text prompt description. We use a bezier representation of the sketch since the sketch contains only the strokes. Since previous methods are not able to estimate the smooth and accurate motion, they also fail to preserve the topology of the sketch in the animation. We use LA regularization to maintain temporal consistency by estimating smooth and accurate motion. Further, a shape preserving arap loss to maintain the rigidity of the sketch helps to avoid the topology change and shape distortion. Our method outperforms state-of-the-art results in both quantitative and qualitative evaluation.
